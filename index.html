<script>
   document.addEventListener("DOMContentLoaded", function() {
       const observerOptions = {
           root: null,
           rootMargin: "0px",
           threshold: 0.1 /* 10% do elemento visível */
       };

       const observer = new IntersectionObserver((entries, observer) => {
           entries.forEach(entry => {
               // Se o elemento estiver visível na viewport
               if (entry.isIntersecting) {
                   entry.target.classList.add('visible');
                   // Opcional: Para executar o fade-in apenas uma vez, descomente a linha abaixo
                   // observer.unobserve(entry.target);
               }
               // Opcional: Se quiser que o fade-in se repita ao rolar para cima e para baixo, descomente o bloco 'else'
               /* else {
                   entry.target.classList.remove('visible');
               } */
           });
       }, observerOptions);

       // Aplica o observer a todos os elementos com a classe 'fade-in-on-scroll'
       document.querySelectorAll('.fade-in-on-scroll').forEach(element => {
           observer.observe(element);
       });

       // Adiciona a classe 'visible' aos elementos que já estão na viewport no carregamento
       observer.takeRecords().forEach(entry => {
           if (entry.isIntersecting) {
               entry.target.classList.add('visible');
           }
       });
   });
</script>

Detalhes do Design Implementado:

Funcionalidade/Estilo

Implementação no Código

Paleta Terracota

Variáveis CSS (:root) usando os tons da imagem: --cor-primaria (Poppy/Terracota), --cor-secundaria (Velvet/Bordô), --cor-destaque (Pampas/Ocre), etc.

Fontes

Definidas nas variáveis CSS: --fonte-titulo (Courier New), --fonte-corpo (Century Gothic com fallback para Montserrat).

Elemento Musical/Circular

O <div class="logo-simbolo">♪</div> no cabeçalho cria um elemento circular (com border-radius: 50%) e usa o símbolo "♪" (nota musical) para remeter aos instrumentos/podcast.

Podcast/Vídeo

Adicionados links explícitos no menu (🎙️ Podcast e ▶️ Vídeos) para destacar essas seções.

Fade de Baixo para Cima

Estrutura CSS (.fade-in-on-scroll) e um pequeno script JavaScript para acionar a transição de transform: translateY(50px) e opacity: 0 para transform: translateY(0) e opacity: 1 quando o elemento é visível.
